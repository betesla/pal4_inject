PAL4 Inject v0.1.5

## 更新重点
- 新增宽屏 UI 支持。CEGUI 渲染、鼠标坐标、小地图、头像、战斗提示、战斗胜利图与存档列表等界面现在会按宽屏方案重新布局，减少 16:9 / 16:10 下 UI 被挤在中间或位置错误的问题。
- 改进高清文本渲染实验。对白、系统字体与按钮标题的高清化链路继续完善，并修复部分多行说明文字行距偏大的问题。
- 新增手柄 / XInput 实验功能。可在控制面板中启用手柄输入，并带有按钮路由、摇杆保持状态修复和使用说明文档。
- 新增阴影质量控制。可调整 PAL4 的阴影贴图尺寸、中间纹理与模糊次数，用于在画质和性能之间取舍。
- 新增实验性坐姿 VR 相机管线。该功能仍处于实验阶段，适合测试和反馈，不建议当作稳定默认功能。
- 新增 UI 纹理过滤选项。可按偏好选择更平滑或更接近原版像素感的界面纹理显示效果。
- 新增 IDA Hex-Rays 伪代码批量导出脚本，方便后续分析 `CEGUIBase.dll`、`OIRAMLOOK.dll` 等模块。

## 修复与改进
- 修复读档 / 存档列表中截图缩略图在宽屏渲染路径下可能被背景覆盖的问题。
- 修复宽屏小地图纹理仍按居中 4:3 区域定位的问题，现在会跟随左下角 HUD 框体。
- 修正战斗 HUD 坐标处理，避免部分战斗图标、浮动数字、胜利图在宽屏下被重复缩放或偏移。
- 修复战斗 HUD 头像在宽屏资源中可能横向拉伸的问题。
- 修复手柄左摇杆保持状态与按钮路由稳定性问题。
- 改进启动器声明页、控制面板说明、部署说明和默认输入配置。
- 精简过时 hook 入口，补充 hook inventory、架构说明、宽屏和文本渲染调查文档。
- 修复 Gitee Release 更新路径，提升发布脚本对 GitHub / Gitee 双平台发布的稳定性。

## 安装方式
1. 下载并解压 `PAL4_inject_v0.1.5_win32.zip`。
2. 将解压后的所有文件复制到 PAL4 游戏安装目录，使 `PAL4_inject.exe` 与 `PAL4.exe` 位于同一目录。
3. 保持 `pal4_inject` 文件夹与 `PAL4_inject.exe` 同级，不要只复制 exe。
4. 双击 `PAL4_inject.exe` 启动游戏。

## 使用说明
- 启动器默认使用 `CSB` 模式。
- 控制面板默认隐藏，按 `Ctrl+J` 可显示 / 隐藏。
- 运行产生的配置、日志和 crash 文件会保存在 `pal4_inject` 子目录。
- 手柄功能、阴影质量、宽屏 UI、高清文本、VR 相机等仍建议按需启用并保留原始游戏备份。
- 控制面板详细说明见仓库文档：
  - `docs/control_panel_guide.md`
  - `docs/gamepad_guide.md`
  - `docs/gameplay_controls.md`
