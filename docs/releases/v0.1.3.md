PAL4 Inject v0.1.3

## 更新内容
- 启动器与注入运行时的配置、日志、crash report / minidump 现在统一放到游戏目录下的 `pal4_inject` 子目录，便于覆盖更新和集中管理。
- 控制面板默认配置已调整：
  - “渲染与画面”相关修复默认启用。
  - 所有 hook 日志默认关闭，减少普通玩家使用时的日志噪音。
  - “对白文本注入”默认关闭，避免直接替换游戏对白。
- 修复更新检查在 Gitee Release 已存在时可能跳错页面的问题，现在会优先使用正确的 release 页面地址。
- 新增中文《控制面板使用说明》，更方便向普通玩家解释每个开关的作用。

## 安装方式
1. 下载并解压 `PAL4_inject_v0.1.3_win32.zip`。
2. 将解压后的所有文件复制到 PAL4 游戏安装目录，使 `PAL4_inject.exe` 与 `PAL4.exe` 位于同一目录。
3. 保持 `pal4_inject` 文件夹与 `PAL4_inject.exe` 同级，不要只复制 exe。
4. 双击 `PAL4_inject.exe` 启动游戏。

## 使用说明
- 启动器默认使用 `CSB` 模式。
- 控制面板默认隐藏，按 `Ctrl+J` 可显示 / 隐藏。
- 运行产生的配置、日志和 crash 文件会保存在 `pal4_inject` 子目录。
- 控制面板详细说明见仓库文档：
  - `docs/control_panel_guide.md`
